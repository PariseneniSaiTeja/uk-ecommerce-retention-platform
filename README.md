# UK E-Commerce Customer Retention & Revenue Analytics Platform

## Overview
This project builds a cloud-based analytics and machine learning platform to analyse customer behaviour, retention, and revenue for a UK e-commerce business.

The platform delivers executive dashboards, customer segmentation, and churn prediction models to support data-driven retention and growth strategies.

## Business Objectives
- Analyse customer purchase behaviour and revenue trends
- Measure customer retention and churn
- Identify high-value and at-risk customer segments
- Predict customer churn to support targeted retention strategies
- Provide executive-level insights through interactive dashboards

## Tech Stack
- SQL (BigQuery)
- Python (pandas, scikit-learn)
- Power BI
- Cloud Data Warehouse
- GitHub for version control

## Key Deliverables
- Cleaned and modelled analytics tables
- Executive KPI dashboards
- Customer segmentation models
- Churn prediction models
- Business insights and recommendations report
